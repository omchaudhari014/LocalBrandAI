🎨 YOUR BACKEND CODE:

const express = require("express");
const cors = require("cors");
const app = express();

app.use(cors());
app.use(express.json());

app.post("/generate", (req, res) => {
  const { businessType, offer, language } = req.body;
  const caption = `Special ${offer} at our ${businessType}! Visit today! (${language})`;
  res.json({ message: caption });
});

app.listen(5000, () => {
  console.log("Server running on port 5000");
});

🔧 TROUBLESHOOTING:

Problem: "npm is not recognized"
Fix: Install Node.js from https://nodejs.org/

Problem: "Cannot find module 'express'"
Fix: Run "npm install express cors" in localbrandai-backend

Problem: "Failed to connect to server"
Fix: Make sure backend is running (port 5000)

Problem: "Port 5000 already in use"
Fix: Change port in server-simple.js to 3000 or 4000
