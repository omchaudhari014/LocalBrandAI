🎯 TWO OPTIONS TO USE DATABASE:

┌─────────────────────────────────────────────────────────────┐
│ OPTION 1: Local MongoDB (Offline)                          │
└─────────────────────────────────────────────────────────────┘

1. Install MongoDB Community Server
   → https://www.mongodb.com/try/download/community
   
2. Install dependencies:
   cd localbrandai-backend
   npm install mongoose

3. Start server:
   node server-with-db.js

4. Open: index-with-database.html

┌─────────────────────────────────────────────────────────────┐
│ OPTION 2: MongoDB Atlas (Cloud - Free)                     │
└─────────────────────────────────────────────────────────────┘

1. Create free account at:
   → https://www.mongodb.com/cloud/atlas/register

2. Create cluster (takes 3-5 minutes)

3. Get connection string

4. Update .env file:
   MONGODB_URI=mongodb+srv://user:pass@cluster.mongodb.net/localbrandai

5. Install dependencies:
   npm install mongoose

6. Start server:
   node server-with-db.js

🔧 TROUBLESHOOTING:

Problem: "Cannot find module 'mongoose'"
Fix: npm install mongoose

Problem: "MongoDB connection error"
Fix: Make sure MongoDB is running
     For local: net start MongoDB
     For Atlas: Check connection string

Problem: Database shows "Disconnected"
Fix: Check .env file has MONGODB_URI
     Verify MongoDB service is running
